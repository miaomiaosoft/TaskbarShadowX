# TaskbarShadowX
为系统任务栏（和顶端）添加阴影效果，可设置阴影大小、位置微调、透明度等，系统美化辅助工具。
网上虽已有同类工具，但都有各种问题，如无法在系统全屏切换中自动调整、阴影层次不跟随任务栏、阴影样式单一等等。
本工具试图解决这些问题！
